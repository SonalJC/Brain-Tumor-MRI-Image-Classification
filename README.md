#  Brain Tumor MRI Image Classification

A deep learning system that classifies brain tumors from MRI scan images using **PyTorch** and **Streamlit**.

---

##  About The Project

Brain tumor detection from MRI scans is a critical medical task that requires high accuracy. This project builds a deep learning model to automatically classify MRI images into tumor categories, helping assist in early diagnosis.

---

##  Results

| Metric | Score |
|--------|-------|
| Test Accuracy | **86.15%** |

---

##  Tech Stack

- **Language:** Python
- **Deep Learning:** PyTorch
- **Web App:** Streamlit
- **Environment:** Jupyter Notebook
- **Libraries:** torchvision, NumPy, Matplotlib, Pillow

---

##  Features

- Upload any brain MRI image and get instant classification
- Deep learning model trained on labeled MRI dataset
- Clean and simple web interface built with Streamlit
- Real-time prediction with confidence score

---

## How To Run

```bash
# 1. Clone the repository
git clone https://github.com/SonalJC/Brain-Tumor-MRI-Image-Classification.git

# 2. Install dependencies
pip install -r requirements.txt

# 3. Run the Streamlit app
streamlit run app.py
```
